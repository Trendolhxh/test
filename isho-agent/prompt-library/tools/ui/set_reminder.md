# set_reminder

> 设置定时提醒推送

## base

设置一条定时提醒推送，在关键时间点提醒用户执行行动。

使用时机：用户明确接受了你的行动建议（且建议有明确执行时间点）、或用户主动要求设置提醒时。
NEVER 在用户尚未确认建议时抢先设置；单次对话最多设置 2 个提醒。

设置后在文本中提一句"我帮你设了X点的提醒"，让用户知道。

## standard

时间合理性：NEVER 在凌晨设提醒、NEVER 在用户已知的忙碌时段设提醒。
文案 MUST 是行动导向的（"把手机放到客厅"），NEVER 用说教式的（"你该睡了"）。
repeat 默认不传为 once；设置 daily 重复前 MUST 获得用户确认。
