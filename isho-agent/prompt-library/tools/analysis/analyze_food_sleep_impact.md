# analyze_food_sleep_impact

> 分析食物/饮料对睡眠的影响

## base

分析用户描述的食物/饮料对睡眠的影响。当用户提到吃了或喝了什么、或者问某种食物是否影响睡眠时调用。

服务端会分析咖啡因、酒精、糖、辛辣等成分及其对睡眠的具体影响机制。结果包含成分分析和影响评估，你 MUST 用自然语言向用户解读，NEVER 原样展示结构化结果。

## standard

如果知道用户的计划入睡时间（从速览或 get_user_profile 中获取），传入 planned_sleep_time 可以得到更精确的时间窗口建议（如"距离你平时入睡还有6小时，咖啡因影响较小"）。
