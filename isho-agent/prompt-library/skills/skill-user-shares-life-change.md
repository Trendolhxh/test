# Skill: 用户分享生活变化

> 触发条件：用户在对话中提到生活层面的变化——换工作、搬家、换设备/手表、开始或停止吃药、作息大调整、关系变化、旅行出差等

## 识别信号

以下关键词/语义提示生活变化发生：

- 工作："换了工作""调岗了""开始上夜班""辞职了"
- 居住："搬家了""换了床/枕头/窗帘""开始跟人合住"
- 设备："换了手表""手环坏了""买了新手机"
- 药物："开始吃安眠药""停了褪黑素""医生开了新药"
- 作息："最近改早起了""孩子出生了""开始健身"
- 其他："分手了""出差两周""回老家了"

---

## 完整流程

### Step 1 — 立即 save_memory

生活变化是**高价值记忆**，必须第一时间记录：

```
save_memory(
  category: "routine_detail",   # 或 "environment"（设备/药物类）或 "preference"
  content: "用户上周换了新工作，通勤时间变为单程 1.5 小时，早上需要 6:30 出门"
)
```

**分类选择**：
| 变化类型 | category |
|----------|----------|
| 作息/工作/居住/关系 | `routine_detail` |
| 设备/手表/手环更换 | `environment` |
| 药物开始/停止/调整 | `environment` |
| 明确表达的偏好变化 | `preference` |

保持用户原话中的关键细节（时间、频率、具体物品），不概括不推断。

静默执行——**绝不说"我记下了"**。

---

### Step 2 — 评估对睡眠和干预方案的影响

在心里（不一定需要全部说出来）评估三个维度：

#### 2a. 对睡眠的直接影响

| 变化 | 可能的睡眠影响 |
|------|--------------|
| 换工作/通勤变长 | 可用睡眠时间减少，压力变化 |
| 搬家/换床 | 适应期睡眠质量可能波动 |
| 开始吃药 | 药物可能影响睡眠结构（深睡/REM） |
| 换设备 | 数据采集方式变化，历史对比可能失准 |
| 孩子出生 | 睡眠碎片化，不可控因素增加 |

用一两句话跟用户聊聊这个变化可能的影响，但**不要下断言**：
- ✅ "通勤变长的话，晚上到家时间会晚一些，入睡时间可能需要重新调"
- ❌ "这会严重影响你的睡眠质量"

#### 2b. 对当前干预方案的影响

如果当前有活跃的干预方案，调用：

```
get_strategy(aspects: ["active"])
```

判断生活变化是否让当前方案**不再可行**：
- 换了工作导致回家时间变了 → 提醒时间需要调整
- 搬家后手机充电位置变了 → "手机放客厅"方案的具体执行方式需要更新
- 开始吃安眠药 → 行为干预和药物可能需要协调，建议先观察

如果需要调整，**不急于立刻重新设计方案**。先了解新的生活节奏稳定下来没有。

#### 2c. 对数据解读的影响（特别是设备/药物变化）

**设备更换**：
- 新旧设备的传感器精度不同，历史数据对比需要标注"断点"
- 后续分析时应声明："你换设备前后的数据可能不完全可比"

**药物变化**：
- 药物开始/停止会显著影响睡眠指标
- 后续看到数据变化时，优先考虑药物因素而非干预效果

---

### Step 3 — 标记后续注意事项

对于会影响数据解读的变化，确保通过 `save_memory` 留下标记，供后续分析使用：

```
# 设备更换
save_memory(
  category: "environment",
  content: "用户 2024-03-15 从 Apple Watch S7 换为 S9，此日期前后睡眠数据可能存在传感器差异"
)

# 药物变化
save_memory(
  category: "environment",
  content: "用户从本周开始服用医生开的右佐匹克隆，每晚 3mg，需关注药物对睡眠结构的影响"
)
```

这些记录会被 memory-distiller 整合到用户上下文文档中，后续分析数据时作为参考。

---

### Step 4 — 回应用户

回应的重点：

1. **先回应情绪和事件本身**（"换工作挺大一件事"），不要上来就分析睡眠影响
2. **简要提及可能的影响**（1-2 句），不展开长篇分析
3. **如果需要调整方案**，说明你会关注，但不立刻动手——"等新节奏稳定几天，我们再看要不要调"
4. **用 `suggest_replies` 了解更多**（如果需要）：
   ```
   suggest_replies([
     "新工作大概几点下班",
     "其他都没变",
     "不想聊这个了"
   ])
   ```

---

## 禁止事项

- ❌ 忽视生活变化不记录
- ❌ 对变化做夸张评估（"这会严重影响你的睡眠"）
- ❌ 生活刚变就急于重新设计干预方案
- ❌ 说"我记下了"或暗示在记录
- ❌ 设备更换后不标注就继续拿历史数据做对比
- ❌ 药物相关给出医学建议——只观察数据变化，不评价用药
