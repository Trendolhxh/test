## 一、产品定位：状态页 + 今日时间轴

首页由两个页面协同组成，一页答"我现在怎么样"，一页答"一天的精力怎么走"。

| 页面 | 核心问题 | 产品角色 | 形态 |
|:---|:---|:---|:---|
| **状态页**（首页） | 我现在怎么样？ | 带情绪色彩的精力状态主视觉 | 默认落地页 |
| **今日时间轴**（二级页） | 一天怎么走？ | 极简能量曲线 + 时间刻度 | 点击状态页主指标区整屏跳转，返回键回状态页 |

状态页统一布局自上而下为五层：**主指标数值 + 短时变化 → 状态标题（拟人化一句话） → 3–4 个关键指标小卡 → 正文解读与建议 → 3 条快捷提问 → 心率入口（可下探身体数据）**。同一骨架通过色彩主题和内容替换承载全天四种状态。

## 二、用户使用场景与四色状态

状态页按当日节律自动切换四种色彩主题，用户无需操作；每种主题在同一骨架下替换标题、指标卡组合与正文：

| 状态 | 触发 | 主指标 | 状态文案（示例） | 小卡组合 |
|:---|:---|:---|:---|:---|
| **绿 · 充沛** | 日间高能区间内 | 今日高能时长（如 7h 29m） | "完美静息换来持久巅峰" | 恢复性睡眠 / 运动时长 / 睡眠总时长 |
| **橙 · 需降速** | 日间偏离健康节律 | 今日高能时长（如 5h 15m） | "神经状态虚浮需降速" | 心率 / 压力 / 咖啡因 |
| **深灰 · 归零过渡** | 当日最后一个高能段结束后 | 今日高能时长 0m | "是时候让神经软着陆了" | 恢复性睡眠 / 运动时长 / 睡眠债 |
| **紫 · 入睡准备度** | 夜间模式 | 入睡准备度百分比 + 推荐入睡时间 | "神经张力尚未完全释放" | 睡眠压力 / HRV / 皮肤温度 / 心率 |

**短时变化基准**：主指标下方统一展示"较 2 小时前"的变化（如 ↑ 1.6h），让用户感知最近的动态而非隔日对比。

**典型使用时刻**：晨起看绿/橙判断今日盘面 → 日间看变化微调行为 → 黄昏看深灰收束 → 睡前看紫色准备度决定上床时机。重要日前夜与熬夜场景通过快捷提问进入 Agent 对话，Agent 建议可一键添加到今日时间轴。

## 三、页面状态流转

### 3.1 状态页：四色自动切换

```
绿 · 充沛 ←→ 橙 · 需降速        （日间，随高能水平与偏离度切换）
         ↓
     深灰 · 归零过渡             （最后一个高能段结束后）
         ↓
     紫 · 入睡准备度             （进入夜间模式）
         ↓
     次日 05:00 后首开 → 回到绿/橙
```

四态共享同一五层骨架，切换时仅主指标、标题、指标卡、正文、快捷提问按状态替换；心率等身体数据入口始终在底部保留。

### 3.2 页面跳转

```
状态页（首页）
   │ 点击主指标区域（今日高能时长 / 入睡准备度）
   ▼
今日时间轴（整屏新页）
   · 极简纵向能量曲线（贝塞尔）
   · 左侧每小时时间刻度
   · 顶部小字回显当日高能时长
   · 当前时间横向虚线 + 圆点
   │ 返回键
   ▼
回到状态页（数据同步刷新）
```

## 四、评审验收锚点

- 状态页主指标数值 + 短时变化 + 色彩主题，3 秒内传达"当前怎么样"
- 四色切换对用户透明：同一骨架、同一交互，仅内容替换
- 睡前紫色调调性为"安心不考核"，快捷提问用中性语言引导下一步
- 状态页与今日时间轴的高能时长数据始终一致
